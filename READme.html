01 / Executive Summary

The Big Picture

StockSense AI is a capstone project to build an AI-powered web application that empowers retail investors to evaluate publicly traded stocks through both fundamental analysis (financial health, valuation, growth) and technical analysis (price patterns, momentum, volume signals). The platform uses large language models, machine learning, and real-time market data APIs to surface actionable intelligence that previously required institutional-grade tools or professional analysts.

The outcome is a single, conversational interface that accepts a stock ticker, autonomously retrieves and analyses all relevant data, and returns a structured investment report with a clear AI-generated recommendation backed by quantitative evidence.

02 / Problem Statement

Why This Problem Matters

Core Problem: Retail investors lack access to the tools, data pipelines, and analytical expertise needed to perform rigorous stock analysis before making investment decisions — leading to emotionally driven or uninformed trades that erode wealth.

Specific Pain Points

1. Information overload. A single stock generates hundreds of data points — earnings reports, SEC filings, price charts, analyst ratings, news — scattered across dozens of sources. Synthesising this manually is infeasible for an individual investor.

2. Domain expertise gap. Concepts like P/E ratio, MACD crossovers, RSI divergence, DCF valuation, and debt-to-equity ratios require significant financial literacy that most retail investors simply don't have.

3. Emotional bias in decision-making. Without an objective, data-driven framework, investors over-weight recent performance (recency bias) and news sentiment, leading to buy-high-sell-low behaviour.

4. No unified analysis platform. Investors must context-switch between charting platforms, financial news aggregators, SEC EDGAR, and spreadsheet models — there is no single tool that unifies the full workflow.

03 / Proposed AI Solution

How StockSense AI Solves It

The application follows an AI Agent architecture where a central LLM orchestrates specialised analysis modules. A user inputs a stock ticker; the system autonomously invokes multiple tools, synthesises their outputs, and delivers a structured report — all within seconds.

01

User Intent & Stock Ingestion

User enters ticker symbol (e.g. AAPL, TSLA). The system resolves company metadata and routes to the appropriate data pipelines.

02

Fundamental Analysis Module

Pulls financial statements (income, balance sheet, cash flow), computes key ratios (P/E, P/B, EV/EBITDA, ROE, Debt/Equity), performs DCF valuation, and compares against sector peers.

03

Technical Analysis Module

Fetches OHLCV price data. Computes indicators: Moving Averages (SMA/EMA), RSI, MACD, Bollinger Bands, Volume trends, support/resistance levels. Classifies chart patterns using ML model.

04

Sentiment & News Intelligence

Aggregates recent news articles, analyst reports, and social signals via web search. NLP sentiment scoring applied. Earnings call transcript key-phrase extraction.

05

AI Synthesis & Recommendation Engine

LLM (Claude API) ingests all module outputs, applies investment frameworks (e.g. Value Investing, Growth, Momentum), resolves conflicting signals, and generates a plain-language investment thesis.

06

Structured Report Output

Delivers: Overall Score (Bull / Neutral / Bear), Fundamental Score, Technical Score, Sentiment Score, Risk Factors, and a conversational explanation the user can interrogate further.

04 / Key Features

Core Application Capabilities

📊
Fundamental Scorecard

Automated extraction and scoring of 30+ financial metrics. Valuation multiples benchmarked against sector median. DCF intrinsic value estimate with margin of safety calculation.

📈
Technical Chart Analysis

Interactive candlestick charts with overlaid indicators. ML-based chart pattern recognition (Head & Shoulders, Cup & Handle, etc.). Trend strength and momentum signals.

Real-time news aggregation with NLP sentiment scoring. Earnings call transcript summaries. Insider trading activity flags. Analyst consensus tracker.

07 / Risk & Ethics

Responsible AI in Finance

Risk Mitigation Strategy
AI Hallucination of Financial Metrics All financial data sourced from APIs, not AI generation. LLM only reasons over verified data. Outputs include source citations.
Misleading Investment Advice Clear disclaimers throughout UI: "Not financial advice." Users are shown raw data supporting every claim. Recommendation framed as research aid, not directive.
Data API Rate Limits / Downtime Redis caching for recent queries. Fallback to secondary data providers. Graceful degradation messaging.
Model Bias toward Recent Market Regimes Multi-timeframe analysis (1M, 3M, 1Y, 5Y). Explicit disclosure of model training period. Human-interpretable explanations for every signal.
Privacy & Data Security No user portfolio data stored without consent. JWT authentication. Data encrypted at rest. GDPR-aligned data handling.

09 / Conclusion

The Value Proposition

StockSense AI bridges the gap between institutional-grade investment analysis and the individual retail investor. By combining the structured rigour of fundamental and technical analysis with the reasoning and language capabilities of modern AI, the platform democratises a workflow that has historically been locked behind expensive terminals and years of financial training.

As a capstone project, it demonstrates end-to-end proficiency in AI agent architecture, API integration, machine learning, financial domain modelling, and full-stack application development — culminating in a product with clear real-world utility and a defensible technical foundation.

Mission: To give every retail investor the analytical power of a professional equity researcher — powered by AI, grounded in data, and explained in plain language.
